This is an admin login system that takes advantage of php sessions and salts passwords going into the database(also decrypts them for checking the login).
2 main pages are included, the login(grants ascess to other restricted pages), registration(as long as you are logged in you can create new admins)
There is an example logout button on registration that is functional. 
Database can be imported into mysql admin
